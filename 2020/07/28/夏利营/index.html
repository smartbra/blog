广州夏令营——天马望远镜介绍

开场白

​ 大家好，这里是广州大学驻上海天马望远镜记者赵漳为您带来的报道。我是今年毕业的广大学生赵漳，刚入职上海天文台。今天就由我给大家介绍一下天马望远镜。

天马介绍

1.概述

​ 望远镜呢是座落在上海松江区天马山旁，其直径为65米而的名的。目前应该式亚洲最大 的单天线射电望远镜。这个望远镜在收纳状态下，高度为70米。总重量为2700吨，是一个拥有八波段的卡塞格林式的射电望远镜。这八个波段分别式L、S、C、X、Ku、K、Ka、Q。覆盖了从1.25GHz——50GHz的频率范围，主要承担了2012年嫦娥二号、2013年嫦娥五号飞行试验器的定轨侧轨任务。同时也成功展开了谱线、脉冲星和VLBI的射电天问观测。产出了一些重大的科研成果，比如我老师最近在自然科学杂志发表的关于在年轻大质量恒星中发现3种新的分子脉泽，有兴趣的可以去了解一下。这个时候各位手机电脑前的老铁们，就可以点波关注，双击666走一波了。

2.火星探测

​ 再讲讲最近天马望远镜在干的工作，大家都知道。在北京时间2020年7月23日，海南文昌航天发射场，使用长征五号遥四运载火箭将我国第一颗火星探测器“天问一号”发射升空。就发射到目前还是比较顺令，而在其中呢。天马望远镜、包括南山、昆明、密云的台站组成VLBI网，对天问火星探测器进行测轨，以确保探测器到达预定的轨道位置。之后“绕着巡”，及“环绕”、“着陆”、“巡视”火星这3项任务，提供准确的为位置保证。

3.谱线

​ 在不执行任务的时候呢，因为天马望远镜式全年无休的，空余时间就会开放给国内的科研工作，用于做谱线观测、脉冲星的观测、还加入了EVN - european VLBI Network ，还参与一些VLBI的联测，其主要目的也是对谱线脉冲星大的观测。

4.环境介绍

​ 左边的是终端室、右边的室观测室。左边这些机架上就是放着天马望远镜的一些终端，包括谱线和脉冲星的记录设备DIBAS、VLBI的记录设备、DBBC3、mak6、还有一些控制设备和主动面、水汽辐射计、全系测量、信息监控等设备。观测室呢，就是主要进行观测、天线控制、信息监控的地方。在执行任务，我就不进去了。

结束

欢乐的时光总是短暂，我的天马望远镜介绍要告一段落了。此情此景，作为天文圈的灵魂诗人我准备赋诗一首。

天问升空很成功，

天马测轨进行中。

师弟师妹勤努力，

未来建设中国梦。

好了，把话筒交还主持人。我就先撤了！
