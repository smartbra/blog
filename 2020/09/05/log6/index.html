简介

OpManager是一个用于企业设备监控、管理企业网络的监控工具。同时较强的可扩展性为未来设备的扩容和更新留出较多的冗余。基于天马望远镜的多设备、多责任人的现实情况而选择的。

添加设备

OpManager使用的检测协议有俩种Snmp（Simple Network Management Protocol）和Wmi（Microsoft Windows Management Instrumentation）。从名字看就知道Wmi 对windows的支持要好一点！

以下是俩种区别

监测项 Snmp Wmi
系统信息主机名称 有 无（手动添加）
接口 有 有
CPU 有 有
内存 有 有
进程 有 有
服务 有 有
TCP端口 有，自动扫描 无，手动添加
监控准确程度 准确 准确

也没太大区别，windows 推荐使用Wmi 而linux 用Snmp 就好了（因为在windows SP2 下没找到Snmp，后查证发现要安装独立安装包），虽然也有在Linux上开启Wmi的骚操作，但是不太推荐。

附上骚操作（本人没实践过）：

1.安装存储库（CentOS6或者更高版本）

2.从资源库安装WMIC。
